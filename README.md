# Git Practice

A clean and well-structured repository for practicing **Data Structures & Algorithms in C++**.  
This repo is designed to help students and enthusiasts **understand, implement, and learn DSA concepts** effectively.

## Tech Stack
- **Languages:** C++ (STL, OOP)  
- **Data Structures:** Arrays, Strings, Stacks, Queues, Trees, Graphs  
- **Algorithms:** Sorting, Searching, Recursion, Dynamic Programming, Greedy
